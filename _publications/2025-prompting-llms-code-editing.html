--- layout: publication year: 2026 title: "Understanding and supporting how developers prompt for LLM-powered code editing in practice" authors: - Daye Nam - Ahmed Omran - Ambar Murillo - Saksham Thakur - Abner Araujo - Marcel Blistein - Alexander Frömmgen - Vincent Hellendoorn - Satish Chandra venue: "DL4C Workshop at ICML 2026" venue_url: https://dl4c.github.io/callforpapers/ type: - Peer-reviewed Conference tags: - Large Language Models - Code Editing - Software Engineering - Developer Tools arxiv: "2504.20196" pdf: https://arxiv.org/pdf/2504.20196 highlight: true --- Large Language Models (LLMs) are transforming software engineering, with coding assistants becoming prevalent. This paper investigates developer interactions with an LLM-powered code editing feature, analyzing telemetry logs, identifying prompt challenges, and proposing AutoPrompter to improve prompts by inferring missing context.
