--- layout: publication year: 2025 title: "An Empirical Study of Developer-Provided Context for AI Coding Assistants in Open-Source Projects" authors: - Shaokang Jiang - Daye Nam type: - Preprint tags: - Software Engineering - Artificial Intelligence - Human-Computer Interaction - AI Coding Assistants arxiv: 2512.18925 doi: 10.48550/arXiv.2512.18925 --- This research examines how developers use machine-readable directives to provide project context to AI coding assistants. Through analysis of 401 open-source repositories, the authors created a taxonomy organizing essential project context into five themes: Conventions, Guidelines, Project Information, LLM Directives, and Examples. The study explores how this contextual information varies across different project types and programming languages.
