--- layout: publication year: 2019 title: "The long tail: Understanding the discoverability of API functionality" authors: - Amber Horvath - Sachin Grover - Sihan Dong - Emily Zhou - Finn Voichick - Mary Beth Kery - Shwetha Shinju - Daye Nam - Mariann Nagy - Brad Myers venue: VL/HCC '19 venue_tags: - VL/HCC type: - Peer-reviewed Conference tags: - Software Engineering - API Usability - Human-Computer Interaction - Discoverability pdf: http://www.cs.cmu.edu/~marmalade/papers/Horvath-VL-HCC-2019.pdf --- This paper investigates how programmers discover API functionality, focusing on Apache Beam. Key findings include that prior API experience significantly impacts feature discovery and forming a top-down mental model can limit feature exploration.
