--- layout: publication year: 2026 title: "HANSEL: Extracting Breadcrumbs from Web Agent Trajectories for Interactive Verification" authors: - Yujin Zhang - Daye Nam venue: arXiv preprint venue_url: https://arxiv.org type: - Preprint tags: - Human-Computer Interaction - Web Agents - AI Verification - Developer Tools arxiv: "2606.18671" pdf: https://arxiv.org/pdf/2606.18671 highlight: true --- Web agents performing multi-step tasks are difficult to verify. HANSEL extracts interactive, verifiable evidence from web-agent trajectories by identifying evidence pages and snippets as navigable views that preserve page states like filters and search queries, reducing trajectory volume by 61.6% while achieving 83.7% precision and 88.8% recall in evidence identification.
