--- layout: publication year: 2026 title: "Motif: Discovering and Automating Personal Web Workflows" authors: - Shaokang Jiang - Daye Nam venue: arXiv Preprint venue_tags: - arXiv type: - Preprint tags: - Web Automation - Workflow Discovery - Personal Computing - Program Synthesis - HCI pdf: https://arxiv.org/pdf/2607.10531 link: https://arxiv.org/abs/2607.10531 --- Motif is a system that passively monitors browser activity to discover recurring patterns suitable for automation—addressing the key challenge that users often don't know what can be automated. When Motif identifies a recurring pattern, it recommends an automation to the user and generates executable code upon confirmation, which users can refine using natural language. A multi-day study with eight participants showed that Motif discovered more automatable patterns than users recognized on their own, with most discoveries aligning well with participant routines.
